# gym-architect

Architect Pro: Elite 60 — single-file workout web app.

How to use
1. Open `index.html` in the repository (or host with GitHub Pages / Netlify).
2. Add to Home Screen on mobile for an app-like experience.

Files included:
- index.html — main app (PWA meta + improved JS)
- manifest.json — PWA manifest
- icons/* — simple SVG icons
